# Medical Insurance Cost Predictor 🏥

An End-to-End Machine Learning web application that predicts the annual medical insurance cost of an individual based on their demographic and health metrics. The app is built using Python and deployed live via Streamlit Cloud.

You can interact with the live working application here: 
👉 *[Live Web App Link](https://medical-insurance-predictor-app-ml.streamlit.app)*

---

## 📌 Project Overview
Healthcare costs can vary drastically from person to person. This project analyzes key features that influence insurance premiums and leverages a trained Machine Learning model to provide instant, data-driven cost estimates. 

Key parameters considered for prediction:
* *Age:* Current age of the individual.
* *Gender:* Male or Female.
* *BMI (Body Mass Index):* To check if the individual falls under a healthy weight range.
* *Children:* Total number of dependents covered under the insurance.
* *Smoker Status:* Whether the user smokes or not (one of the highest contributing factors).
* *Region:* Geographic residential area.
